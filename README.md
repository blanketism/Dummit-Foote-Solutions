# Solutions to Dummit and Foote

An unofficial, community-driven LaTeX project for:

- worked solutions to selected exercises from *Abstract Algebra, 3rd Edition* by Dummit and Foote,
- chapter notes and supporting writeups,
- small coding experiments related to algebra topics.

This repository is an independent project and is not affiliated with the textbook authors or publisher.

## What Is In This Repository

The project currently has two primary compilable documents:

- `dummit_foote_exercises.tex`: the main solutions document.
- `dummit_foote_notes.tex`: chapter notes document. **EDIT 7/13/2026** - the notes files are currently defunct due to changes in `base-tex-files/`. I have done a design overhaul for the exercises and will do something similarly for the notes.

Shared macros and environments live in `base-tex-files/` and are reused across files.

## Repository Layout

High-level structure:

- `dummit_foote_exercises.tex` - main entry point for the solutions PDF.
- `dummit_foote_notes.tex` - main entry point for notes PDF.
- `exercises/` - exercise solutions organized by chapter and section files.
- `notes/` - chapter-level notes files.
- `extra-text/` - front-matter style content (preface, contributors).
- `base-tex-files/` - reusable macros, commands, environments.
- `coding-questions/` - Python scripts for computational checks and small explorations.

Auxiliary files (`.aux`, `.toc`, `.fls`, `.fdb_latexmk`) are generated by LaTeX tools and may appear after builds.
## Contributing

Contributions are welcome.

1. Fork the repository.
2. Create a branch for your change.
3. Make edits with clear commit messages.
4. Ensure the relevant `.tex` files compile.
5. Open a pull request.

Suggested contribution style:

- keep notation consistent with surrounding material,
- prefer reusable macros in `base-tex-files/` for repeated symbols or structures,
- keep chapter files focused and easy to review.

## Notes On Scope and Attribution

This is an ongoing educational project intended to support learning. Please see the preface material in the compiled exercises document for additional context.
